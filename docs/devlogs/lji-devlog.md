  - Named Cloudflare Tunnel(도메인 필요) · ngrok 같은 대안도 검토했지만 지금 단계에서 무료로 안정적인 도메인/터널을 구하기가 마땅치 않아서, 일단 Quick Tunnel을 그대로 쓰고 화질(해상도·비트레이트)을 낮추는 쪽으로 대응하기로 함 — 화질을 단계적으로 올려가며 버퍼링 안 생기는 절충점을 찾아 확정, 프론트엔드 버퍼 임계값도 같이 재조정해서 초반 대기 시간도 줄임

  - `media_script`는 위원 전원을 다 채우고 "실제 아바타 영상을 만들지 말지"는 프론트(`CommitteeVideoStage`)에서만 걸러내기로 함 — 처음엔 백엔드에서 아바타 있는 위원만 걸러서 채웠다가, 텍스트 채팅(`MeetingChat.jsx`)까지 같이 줄어드는 부작용을 발견해서 되돌림
  - 다른 담당자 파일(경이님, 가은님)을 건드릴 때마다 파일 단위로 미리 허락을 받고 진행 — 회의 분석 파이프라인과 화면 흐름을 잘 아는 담당자 확인 없이는 구조를 바꾸지 않기로 함
